# SelfLearning

This is my personal learning repository - a place to store and organize what I have learnt.

## Purpose

This repository serves as a centralized location to document my learning journey across various topics, technologies, and concepts. It helps me:
- Keep track of new skills and knowledge
- Create a reference for future use
- Organize learning materials systematically
- Track my progress over time

## Repository Structure

```
SelfLearning/
├── programming/          # Programming languages and concepts
├── frameworks/           # Web frameworks, libraries, and tools
├── databases/            # Database technologies and queries
├── algorithms/           # Algorithms and data structures
├── devops/              # DevOps tools and practices
├── concepts/            # General CS concepts and theories
└── resources/           # Books, courses, and learning resources
```

## How to Use

1. Create a new folder for each topic or category
2. Add markdown files with notes, code snippets, and examples
3. Include references and links to learning resources
4. Keep content organized and easy to navigate

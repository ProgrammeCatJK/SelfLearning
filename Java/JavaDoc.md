# Java Annotation + Documentation

## Comments in Java
- `//` → Single line comment  
- `/* ... */` → Multiple line comment  
- `/** ... */` → Documentation comment, used for **Javadoc**  

### Example of Javadoc
```java
/**
 * Search「JK」，Return xdd
 * @author JK xdd
 * @date ${21 Jan}
 */
```

---

## Creating Javadoc
1. Open your project folder in terminal
2. Run the command:
```bash
javadoc filename.java -encoding utf-8
```
3. Check output:
```bash
ls -l
```
4. Open `index.html` in a browser to view documentation
